# TensorFlow Lite Android Experiments

This repository contains on-device ML Android applications
cloned from open-source examples and modified for learning.

## Work Done
- Resolved Gradle compatibility issues
- Configured modules to run on emulator/device
- Added custom documentation
- Tested image classification and detection samples

## Environment
- Android Studio
- Kotlin
- TensorFlow Lite

## Run
Open any example module in Android Studio and deploy on
Android emulator or physical Android device.
